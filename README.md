# Projet LO07 - BlaBlaCar

A PHP/MySQL MVC web application inspired by BlaBlaCar, developed for the LO07 Web Technologies course at UTT.

## Features

- User authentication and logout
- Role-based access control for administrators, drivers, and passengers
- User management
- Vehicle management
- City management
- Trip creation, listing, and closing
- Trip reservation management
- MySQL database integration
- MVC architecture with controllers, models, and views

## Technologies

- PHP
- MySQL
- HTML
- CSS
- MVC architecture
